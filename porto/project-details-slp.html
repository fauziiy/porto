Pengenalan Pola Menggunakan SLP

Deskripsi Proyek

Proyek ini berfokus pada penerapan model Single Layer Perceptron (SLP) untuk melakukan klasifikasi pada dataset numerik seperti Iris dan MNIST. SLP merupakan model jaringan syaraf tiruan sederhana yang digunakan untuk memisahkan data yang linearly separable. Proyek ini mencakup preprocessing data, pelatihan model, dan evaluasi performa model menggunakan metrik seperti akurasi dan confusion matrix.

Teknologi yang Digunakan

  • Python
  • Scikit-learn
  • Matplotlib
  • Jupyter Notebook

Hasil dan Visualisasi

Model SLP berhasil mencapai akurasi hingga 95% pada dataset Iris dan 85% pada dataset MNIST setelah tuning parameter seperti learning rate dan jumlah iterasi. Visualisasi dilakukan menggunakan Matplotlib untuk menampilkan decision boundary dan confusion matrix.
