Klasifikasi Citra Menggunakan CNN

Deskripsi Proyek

Proyek ini membangun sistem klasifikasi citra berbasis Convolutional Neural Network (CNN) untuk mendeteksi jenis pemandangan dari dataset gambar. Proyek ini mencakup tahap augmentasi data untuk meningkatkan generalisasi model, tuning arsitektur CNN seperti jumlah lapisan konvolusi dan filter, serta evaluasi performa menggunakan metrik seperti akurasi dan F1-score.

Teknologi yang Digunakan

  • Python
  • TensorFlow
  • Pandas
  • OpenCV

Hasil dan Visualisasi

Model CNN mencapai akurasi rata-rata 92% pada dataset uji. Visualisasi hasil mencakup contoh prediksi citra dan heatmap dari lapisan konvolusi untuk memahami area fokus model.
