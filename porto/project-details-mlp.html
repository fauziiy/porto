Pengenalan Pola Menggunakan MLP

Deskripsi Proyek

Proyek ini menerapkan Multi Layer Perceptron (MLP) untuk klasifikasi dataset numerik seperti Iris dan MNIST. MLP adalah model jaringan syaraf tiruan dengan beberapa lapisan tersembunyi yang mampu menangani data yang tidak linearly separable. Proyek ini melibatkan optimasi arsitektur model, seperti jumlah neuron dan lapisan, serta penggunaan fungsi aktivasi seperti ReLU dan sigmoid.

Teknologi yang Digunakan

  • Python
  • Scikit-learn
  • Matplotlib
  • Jupyter Notebook

Hasil dan Visualisasi

MLP mencapai akurasi hingga 97% pada dataset Iris dan 90% pada dataset MNIST. Visualisasi hasil berupa grafik loss dan akurasi selama pelatihan, serta confusion matrix untuk evaluasi performa model.
