Prediksi Harga Minyak dengan RNN

Deskripsi Proyek

Proyek ini mengembangkan model prediksi harga minyak mentah menggunakan Recurrent Neural Network (RNN) dengan arsitektur LSTM. Data historis harian digunakan untuk melatih model dalam memprediksi fluktuasi harga. Proyek ini mencakup preprocessing data seperti normalisasi dan pembentukan jendela waktu, serta evaluasi menggunakan metrik seperti Mean Squared Error (MSE).

Teknologi yang Digunakan

  • Python
  • Keras
  • Pandas
  • Matplotlib
  • NumPy

Hasil dan Visualisasi

Model RNN-LSTM menghasilkan prediksi dengan MSE di bawah 0.02 pada data uji. Visualisasi hasil menampilkan perbandingan harga aktual dan prediksi, serta grafik loss selama pelatihan.
