Implementasi Logika Fuzzy untuk Prediksi Risiko Diabetes

Deskripsi Proyek

Proyek ini mengembangkan sistem prediksi risiko diabetes menggunakan logika fuzzy Mamdani. Sistem ini memproses parameter medis seperti tekanan darah, kadar glukosa, dan indeks massa tubuh (BMI) untuk menghasilkan prediksi tingkat risiko diabetes. Antarmuka web interaktif dibangun dengan Flask untuk memudahkan pengguna memasukkan data dan melihat hasil prediksi secara real-time.

Teknologi yang Digunakan

  • Python
  • Scikit-Fuzzy
  • Flask
  • HTML/CSS

Hasil dan Visualisasi

Sistem berhasil memprediksi risiko diabetes dengan tingkat keakuratan yang divalidasi oleh pakar medis. Visualisasi mencakup grafik fungsi keanggotaan fuzzy dan hasil prediksi dalam bentuk persentase risiko pada antarmuka web.
