Deteksi Objek Rambu Lalu Lintas dengan YOLOv8

Deskripsi Proyek

Proyek ini mengembangkan sistem deteksi objek berbasis YOLOv8 untuk mengidentifikasi rambu lalu lintas secara real-time dari input gambar, video, atau kamera. Sistem ini dilengkapi dengan antarmuka web menggunakan Flask untuk memvisualisasikan hasil deteksi dan menyediakan rekap data seperti jenis rambu dan tingkat kepercayaan deteksi.

Teknologi yang Digunakan

  • Python
  • YOLOv8 (Ultralytics)
  • OpenCV
  • Flask

Hasil dan Visualisasi

Sistem YOLOv8 mencapai mAP@50 sebesar 0.85 pada dataset rambu lalu lintas. Hasil deteksi divisualisasikan dengan bounding box dan label pada gambar atau video input, serta tabel rekap deteksi pada antarmuka web.
