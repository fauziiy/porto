Tentang Saya

Saya adalah mahasiswa aktif Program Studi Informatika di Universitas Teknologi Yogyakarta yang memiliki minat mendalam pada bidang Machine Learning dan Artificial Intelligence (AI). Ketertarikan saya berawal dari keingintahuan terhadap bagaimana sistem cerdas dapat meniru cara berpikir manusia dan membantu menyelesaikan berbagai permasalahan kompleks di dunia nyata.

Selama perkuliahan, saya telah mengikuti berbagai proyek dan penelitian yang berkaitan dengan machine learning, di antaranya implementasi model jaringan syaraf tiruan seperti SLP, MLP, CNN, dan RNN. Selain itu, saya juga pernah mengembangkan sistem deteksi objek berbasis YOLOv8, serta sistem pakar berbasis Fuzzy Logic untuk diagnosis penyakit, menggunakan Python dan framework Flask.

Saya senang belajar hal-hal baru yang dapat mengembangkan wawasan dan keterampilan saya dalam bidang data science, pengolahan citra, dan pemrograman AI. Dengan latar belakang ini, saya memiliki tujuan untuk berkontribusi dalam pengembangan teknologi AI yang bermanfaat dan etis, serta terus mengasah kemampuan melalui proyek, kolaborasi, dan penelitian.
